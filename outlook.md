# Tool: Outlook

## Problem 1: Cannot sign in
**Solution:**
This happens when credentials are incorrect or the profile is corrupted.
Steps:
1. Re-enter your password carefully
2. Check Caps Lock and keyboard language
3. Repair Outlook profile from Control Panel
4. Restart Outlook

---

## Problem 2: Can't send messages
**Solution:**
This issue usually occurs due to large attachments or incorrect mail settings.
Steps:
1. Check if the email is stuck in Outbox
2. Reduce attachment size
3. Verify SMTP settings
4. Restart Outlook

---

## Problem 3: Emails not receiving
**Solution:**
Emails may not sync due to internet or filter issues.
Steps:
1. Check internet connection
2. Click Send/Receive
3. Check Junk and Other folders
4. Remove unnecessary rules

---

## Problem 4: Outlook is very slow
**Solution:**
Large mailbox size or add-ins can slow Outlook.
Steps:
1. Disable unused add-ins
2. Archive old emails
3. Compact Outlook data file
4. Restart Outlook

---

## Problem 5: Outlook keeps crashing
**Solution:**
Corrupted add-ins or profiles cause crashes.
Steps:
1. Open Outlook in Safe Mode
2. Disable add-ins
3. Create a new Outlook profile
4. Update Outlook

---

## Problem 6: Attachments not opening
**Solution:**
This happens due to blocked file types or security settings.
Steps:
1. Save attachment to desktop first
2. Ensure correct application is installed
3. Check antivirus restrictions
4. Restart Outlook

---

## Problem 7: Calendar not syncing
**Solution:**
Sync issues occur due to account or permission problems.
Steps:
1. Refresh calendar
2. Check account sync settings
3. Sign out and sign in again
4. Restart Outlook

---

## Problem 8: Search not working
**Solution:**
Outlook search depends on indexing.
Steps:
1. Restart Outlook
2. Rebuild Windows search index
3. Ensure Outlook is indexed
4. Update Outlook

---

## Problem 9: Emails going to Junk folder
**Solution:**
Spam filters may incorrectly classify emails.
Steps:
1. Mark email as “Not Junk”
2. Add sender to Safe Senders list
3. Review Junk Email settings

---

## Problem 10: Outlook stuck on loading profile
**Solution:**
This occurs due to corrupted profiles or network issues.
Steps:
1. Restart computer
2. Disable add-ins
3. Create a new Outlook profile
4. Check internet connection

---

## Problem 11: Cannot add email account
**Solution:**
Incorrect server or authentication settings cause this.
Steps:
1. Verify email address and password
2. Check server settings with IT
3. Try manual account setup
4. Restart Outlook

---

## Problem 12: Duplicate emails appearing
**Solution:**
Sync or rule issues can cause duplicates.
Steps:
1. Check email rules
2. Disable multiple accounts syncing same mailbox
3. Restart Outlook

---

## Problem 13: Outlook not opening
**Solution:**
Application or profile corruption can prevent opening.
Steps:
1. Open Outlook in Safe Mode
2. Repair Office installation
3. Create a new profile
4. Restart system

---

## Problem 14: Emails delayed in sending
**Solution:**
Network or outbox issues cause delay.
Steps:
1. Check internet connectivity
2. Ensure Outlook is not in Offline mode
3. Clear Outbox
4. Restart Outlook

---

## Problem 15: Missing emails
**Solution:**
Emails may be archived or filtered.
Steps:
1. Search All Mailboxes
2. Check Archive folder
3. Review email rules
4. Check Deleted Items

---

## Problem 16: Cannot access shared mailbox
**Solution:**
Permission or sync issues may block access.
Steps:
1. Confirm mailbox permissions with IT
2. Restart Outlook
3. Re-add shared mailbox
4. Sign out and sign in again

---

## Problem 17: Outlook showing offline status
**Solution:**
Offline mode or network problems cause this.
Steps:
1. Turn off Offline mode
2. Check internet connection
3. Restart Outlook
4. Restart computer

---

## Problem 18: Signature not appearing
**Solution:**
Signature settings may not be applied.
Steps:
1. Check default signature settings
2. Ensure correct email account selected
3. Restart Outlook

---

## Problem 19: Cannot delete emails
**Solution:**
Mailbox permissions or sync issues may prevent deletion.
Steps:
1. Restart Outlook
2. Empty Deleted Items folder
3. Repair Outlook data file

---

## Problem 20:  Outlook asking for password repeatedly
**Solution:**
Credential cache or authentication issues cause this.
Steps:
1. Update saved credentials
2. Clear Windows Credential Manager
3. Restart Outlook
4. Contact IT if issue persists
